# Monday Coffee - SQL Data Analysis Project

## 📋 Project Overview
A comprehensive SQL analysis project with 10 business questions solved using PostgreSQL. This project analyzes coffee sales data across multiple cities.

## 🛠 Skills Demonstrated
- SQL Fundamentals (SELECT, WHERE, GROUP BY, HAVING)
- JOINs (INNER, LEFT)
- Aggregate Functions (SUM, COUNT, AVG, MAX)
- Common Table Expressions (CTEs)
- Window Functions (DENSE_RANK, LAG)
- Date Functions (EXTRACT)
- Subqueries

## 📊 Queries Included

| # | Query | Description |
|---|-------|-------------|
| Q1 | Coffee Consumers Count | Estimated coffee consumers (25% population) per city |
| Q2 | Total Revenue | Revenue from coffee sales in Q4 2023 |
| Q3 | Product Sales | Units sold for each coffee product |
| Q4 | Avg Sales per City | Average sales amount per customer by city |
| Q5 | City Population | Population vs actual customers |
| Q6 | Top Products by City | Top 3 selling products in each city |
| Q7 | Customer Segmentation | Unique customers per city |
| Q8 | Sales vs Rent | Average sale and rent comparison |
| Q9 | Monthly Growth | Month-over-month sales growth percentage |
| Q10 | Market Potential | Top 3 cities by revenue with metrics |
